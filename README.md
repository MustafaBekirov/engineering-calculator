# Engineering Calculator

Первый инженерный калькулятор для расчёта `l`.

## Формула

`l = (ωan × Rs / Rb + Δλan) × d`

## Что автоматизировано

- `Rs` по классу и диаметру арматуры, включая A600 = 510 МПа.
- `Rb` по классу бетона от B3.5 до B60.
- `Δλan` по классу арматуры и зоне работы:
  - растянутая зона: 11
  - сжатая зона: 8
- `ωan` по классу арматуры и зоне работы:
  - A240, растянутая зона: 1.55
  - A240, сжатая зона: 1.0
  - A300/A400/A500/A600, растянутая зона: 0.90
  - A300/A400/A500/A600, сжатая зона: 0.65

## Что пока вводится вручную

На этом этапе вручную вводится только диаметр/класс арматуры, класс бетона, зона работы и шаг округления.
`Rb`, `Rs`, `ωan` и `Δλan` подставляются автоматически.

## v13

Добавлены итоговые проверки:

1. `l ≥ λan × d`

Значения `λan`:
- A240/A300/A400/A500/A600, растянутая зона: `20`
- A240/A300/A400/A500/A600, сжатая зона: `15`

2. `l ≥ lmin`

Минимальная длина:
- A240/A300/A400/A500/A600, растянутая зона: `250 мм`
- A240/A300/A400/A500/A600, сжатая зона: `200 мм`

Порядок расчёта:
1. Считается `l` по основной формуле.
2. Считается `λan × d`.
3. Берётся максимум из:
   - `l` по основной формуле;
   - `λan × d`;
   - `lmin`.
4. Если выбран учет сейсмических условий — результат умножается на `1.3`.
5. После этого выполняется округление вверх.

## v14

Добавлен выбор варианта округления итогового значения:

- без округления;
- округлить вверх до 5 мм;
- округлить вверх до 10 мм.

Порядок расчёта закреплён:

1. Считается `l` по основной формуле.
2. Проверяется `λan × d`.
3. Проверяется `lmin`.
4. Берётся максимум из трёх значений.
5. Если выбран учет сейсмических условий, максимум умножается на `1.3`.
6. Только после этого применяется выбранный вариант округления.

То есть при сейсмике калькулятор не округляет `l` до умножения на `1.3`.
