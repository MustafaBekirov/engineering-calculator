# Инженерный калькулятор

Стартовый сайт-калькулятор для технических расчётов.

## Модуль 1

Расчёт длины нахлёста арматуры:

`lan = (ωan × Rs / Rb + Δλan) × d`

В этой версии:

- `Rs` определяется автоматически по классу и диаметру арматуры;
- `Rb`, `ωan`, `Δλan` вводятся вручную;
- результат округляется вверх до заданного шага.

## Файлы

- `index.html` — поля ввода и текст страницы;
- `style.css` — оформление;
- `script.js` — расчётная логика.
