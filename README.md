# Инженерный калькулятор

Стартовый сайт на HTML/CSS/JavaScript для технических расчётов.

## Файлы

- `index.html` — структура страницы
- `style.css` — оформление
- `script.js` — формулы и логика расчёта

## Как открыть локально

Откройте файл `index.html` двойным кликом в браузере.

## Как опубликовать на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него файлы `index.html`, `style.css`, `script.js`, `README.md`.
3. Откройте Settings → Pages.
4. В Source выберите Deploy from a branch.
5. Branch: `main`, folder: `/root`.
6. Нажмите Save.
