Исходные данные

Заполните данные и нажмите «Рассчитать».

Что уже автоматизировано

  • Rs определяется автоматически по классу и диаметру арматуры.
  • ωan для A400/A500 определяется автоматически по зоне работы арматуры.
  • Rb определяется автоматически по выбранному классу бетона.
  • Δλan определяется автоматически по зоне работы арматуры.
  • Проверяется условие l ≥ λan × d.
  • Проверяется минимальная длина lmin в миллиметрах.
  • При выборе сейсмических условий итоговая длина увеличивается на 30%.
