Технические расчёты

Инженерный калькулятор

Первый модуль: расчёт длины нахлёста арматуры по формуле lan = (ωan × Rs / Rb + Δλan) × d.

Расчёт длины нахлёста

Выберите класс и диаметр арматуры. Значение Rs программа определит автоматически. Все длины — в мм, сопротивления — в МПа.

Результат

Заполните поля и нажмите «Рассчитать».

Принятая формула в этой версии

lan = (ωan × Rs / Rb + Δλan) × d

В этой версии Rs определяется автоматически по классу и диаметру арматуры. Значения Rb, ωan и Δλan пока вводятся вручную. Когда будут таблицы коэффициентов, их можно заменить на выпадающие списки.
